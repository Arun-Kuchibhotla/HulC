# Simulations for Statistical Inference for Online Algorithms

In the code provided, we run simulations corresponding to the paper ["Statistical Inference for Online Algorithms" (Carter and Kuchibhotla)](https://arxiv.org/abs/2505.17300). 


There are two types of simulations:
1. [Averaged-SGD (ASGD)](https://github.com/Arun-Kuchibhotla/HulC/tree/main/Online_Algorithms_HulC_simulations/ASGD_sims) (see folder **ASGD_sims**): these simulations use python files.
2. [Additional types of SGD](https://github.com/Arun-Kuchibhotla/HulC/tree/main/Online_Algorithms_HulC_simulations/additional_simulations) (see folder **additional_simulations**): these simulations use R files.

**The results of these simulations are available in a [dynamic Tableau graph](https://public.tableau.com/app/profile/selina.carter6629/viz/OnlineinferencesimulationsOLSandlogisticregression/Coverageandwidthratio_paper).**
